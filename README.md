# Nexfila 🎫

Sistema de gestão inteligente de filas com prioridade para idosos.

Projeto desenvolvido para fins didáticos, documentando o ciclo completo de desenvolvimento:
backend Java/Spring Boot + frontend React/TypeScript + Docker + VPS + HTTPS + CI/CD.

## 🔗 Demo
[https://nexfila.com.br](https://nexfila.com.br)

## 🛠️ Stack
- **Backend:** Java 25 + Spring Boot + Maven
- **Frontend:** React + TypeScript + Vite
- **Infra:** Docker + Nginx + Let's Encrypt + Hostinger KVM 2

## 📚 Sobre
Projeto do curso [Ascendus](https://ascendus.com.br) — do zero ao deploy em produção.
